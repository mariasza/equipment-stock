# API Equipment stock

## Description

```bash
API for equipment stock control
```

## Installation

```bash
$ npm install
```

## Creating database

```bash
$ docker-compose up -d
$ npm run db:create

```

## Running the app

```bash
$ npm run start
```

## Running the tests

```bash
$ npm run test
```
